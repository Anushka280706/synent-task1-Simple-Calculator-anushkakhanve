# Simple CLI Calculator

This is a beginner-friendly Command Line Interface (CLI) Calculator built using Python. The calculator performs basic arithmetic operations such as addition, subtraction, multiplication, and division. It takes user input from the terminal, handles invalid inputs, and displays the calculated results.

## Features
- Addition of two numbers
- Subtraction of two numbers
- Multiplication of two numbers
- Division of two numbers
- Error handling for invalid inputs
- Division by zero handling
- Continuous calculations using loops

## Technologies Used
- Python

## How to Run
1. Save the file as `calculator.py`
2. Open terminal or command prompt
3. Run the following command:

python calculator.py

This project is designed for beginners to practice Python fundamentals like functions, loops, conditional statements, and exception handling.
